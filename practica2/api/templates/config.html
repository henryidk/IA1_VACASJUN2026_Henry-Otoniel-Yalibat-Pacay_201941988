¿Para qué sirve esta sección?

Aquí se configura el grupo de Telegram al que el sistema enviará una notificación automática cada vez que se agregue, edite o elimine una pregunta o categoría desde este panel.

Para activar las notificaciones necesitas el ID del grupo de Telegram. Sigue estos pasos para obtenerlo:

  1. Crea un grupo en Telegram (o usa uno existente) y agrega el bot como miembro.
  2. Envía cualquier mensaje dentro del grupo.
  3. Abre esta URL en el navegador (reemplaza <TOKEN> con el token real del bot):
    https://api.telegram.org/bot<TOKEN>/getUpdates
  4. En el JSON que aparece, busca el campo "chat" y copia el valor de "id". Ese número es el ID del grupo (siempre es negativo, ej: -5107381860).
  5. Pégalo en el campo de abajo y guarda.

Si el campo queda vacío, el sistema simplemente no enviará notificaciones, sin generar ningún error.
